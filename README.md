# futuristic_blogger_themes

Futuristic blogger themes

A repository containing futuristic Blogger themes for a better user-experience

This project is not associated with Blogger team, it is just a side gig, which was created out of the pure boredom and looking into the good old Blogger design.

In other words... I justg had an access to modern AI development tools and had no other ideas. Therefore, I decided to create something useful for the next gen bloggers instead. :D

# How to use the themes (importing)

- Download the desired theme file
- Use the blogger interface -> Restore theme from a file  

[https://bento-san.blogspot.com/](https://bento-san.blogspot.com/)

# Clean Blog-inspired themes

These templates are based on the existing bento Blogger structure, but restyled toward the Start Bootstrap Clean Blog editorial look: large masthead, simple uppercase navigation, readable long-form typography, and high-contrast Material Design color combinations.
